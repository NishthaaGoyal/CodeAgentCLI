# 🚀 CodeAgentCLI

An intelligent CLI-based coding agent powered by LLMs that can **write, read, and execute code directly on your system**.

Built using **LangChain + Ollama (LLaMA3)**, this project demonstrates how AI agents can interact with real-world tools like the file system and terminal.

---

## ✨ Features

- 🧠 Natural language → code generation
- 📁 Create and modify files automatically
- 📖 Read existing files
- ⚙️ Execute terminal commands
- 🛡️ Safe path handling (prevents root access issues)
- 🔧 Self-healing JSON parsing (handles imperfect LLM outputs)

---

## 🛠️ Tech Stack

- Python 3.10+
- LangChain
- Ollama (LLaMA3)
- Subprocess (for CLI execution)
